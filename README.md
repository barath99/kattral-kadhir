# Kattral Kadhir

Responsive React + Vite website for Kattral Kadhir, an education initiative focused on AI literacy, digital literacy, career exposure, and mentorship.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Deploying to Vercel

Import the repository in Vercel and use the default Vite settings:

- Build command: `npm run build`
- Output directory: `dist`

Update the placeholder Google Form URL in `src/App.jsx` before launch.

## GitHub Pages

This repo includes a GitHub Actions workflow for GitHub Pages and a custom domain file for:

```text
kattralkadhir.org
```
